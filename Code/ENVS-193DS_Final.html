Problem 1

a. Transparent statistical methods

In part 1, my coworker used a linear regression statistical method because it compared one predictor variable as precipitation to predict a continuous response variable as flooded wetland area. In part 2 of the results, my coworker used a Kruskal-Wallis test because they compared a median flooded wetland area with more than 2 water year classification categories (5 total).

b. Figure needed

I think that one figure that could accompany the statement for part 2 would be a box plot. The x-axis would show the predictor variable of 5 water year classification categories as wet, above normal, below normal, dry, and critical drought. The y-axis would show the response variable of flooded wetland area (\(m^2\)). Each individual box plot would show the median, interquartile range and the spread to compare distribution across classifications that align with the Kruskal-Wallis test.

c. Suggestions for rewriting

The flooded wetland area did not have evidence of having a relationship with precipitation. This would suggest that precipitation on its own is not a good predictor variable to predict the wetland flooding area in the San Joaquin River Delta (linear regression, test statistic = N/A, df = N/A, p = 0.11, \(\alpha\) = significance level).

The median flooded wetland area did not differ between the water year classifications. This would suggest that across the wet, above normal, below normal, dry, and critical drought years had similar median flooding conditions (Kruskal-Wallis test, test statistic = N/A, df = N/A, p = 0.12, \(\alpha\) = significance level)

d. Interpretation

Based on the Peterson et al. study and my coworkers analysis, one additional variable that could influence the flooded area of a wetland would be local wetland management action as a categorical predictor variable. This variable might influence wetland flooding because the paper states that wetland management could make decisions on moving, retaining or restricting water on specific areas near the wetlands.

Problem 3. Data analysis

a. Response variable

The 0 and 1’s in the data set represent the response variable if a potential nest site was used by a Mountain Plover. A value of 1 represents that a nest site was “used” by the Mountain Plover shown in the id column, and a value of 0 means that the site was “available” or unused by a Mountain Plover shown in the id column.

b. Purpose of study

For the first variable I found to be Visual obstruction as a continuous variable because it measured vegetation height and density around the nesting site using a Robel pole. As the visual obstruction increases, the probability of a Mountain Plover nest site being used would decrease because they prefer live in areas with lower vegetation so they can see predators better. This information was found in the Methods Adults Mountain Plover Density in the Data collection section were visual obstruction is described as being a measure of vegitation structure.

For the second predictor variable of Distance to prairie dog colony edge it is also a continuous variable that is meant to represent how far a nesting location is from from a prairie colony. As the distance from the colony edge increases, the probability that a Mountain Plover nesting site would decrease or vary depending on habitat conditions because the prairie dog would create an open habitat that the Plover might like. This information was found in the Methods Modeling framework section in table 1 that shows the distance to colony edge in the Prairie dog colony-direct section.

d. Run the models

# Running the models
mopl_clean <- mopl |> # created a new object 
  clean_names() |> # changes column names to lowercase and underscores
  # mutate to change the values to bianary numbers were 1 = used nest site, 0 = available
  mutate(used_bin = if_else(id == "used", 1, 0)) |>
  select(used_bin, edgedist, vo_nest) # select() keeps only the columns selected to be used as response and predictor variables
#Running the Models

# null model: includes only the intercept (no predictors)
m1 <- glm(used_bin ~ 1, data = mopl_clean, family = "binomial")
# model with visual obstruction as the only predictor
m2 <- glm(used_bin ~ vo_nest, data = mopl_clean, family = "binomial")
# model with distance to prairie dog colony edge as the only predictor
m3 <- glm(used_bin ~ edgedist, data = mopl_clean, family = "binomial")
# full model: includes both predictors
m4 <- glm(used_bin ~ vo_nest + edgedist, data = mopl_clean, family = "binomial")

e. Select the best model

The best model as determined by Akaike’s Information Criterion (AIC) was the visual obstruction at the nest as the only predictor of the probability of Mountain plover nest site usage because it had the lowest AIC value compared to all other models.

g. Visualize the model predictions

# call for ggplot to create a plot
pred_data <- data.frame(
  vo_nest = seq(min(mopl_clean$vo_nest),
                max(mopl_clean$vo_nest),
                length.out = 100)
)

pred <- predict(m2,
                newdata = pred_data,
                type = "link",
                se.fit = TRUE)

pred_data$fit <- plogis(pred$fit)
pred_data$lwr <- plogis(pred$fit - 1.96 * pred$se.fit)
pred_data$upr <- plogis(pred$fit + 1.96 * pred$se.fit)
ggplot(data = mopl_clean, # calls for mopl_clean data
       aes(x = vo_nest, # assigns x axis to visual obstruction
           y = used_bin)) + # assigns y axis to binary nest use
  
  geom_point(color = "red2") + # to plot raw data points 
  
  geom_line(data = pred_data, # adds model prediction line
            aes(x = vo_nest, # assigns x and y axis data for line
                y = fit),
            color ="blue3", # changes color of prediction line
            linewidth = 2, # changes line width 
            inherit.aes = FALSE) + # prevents using used_bin
  
  geom_ribbon(data = pred_data, # adds 95% CI around prediction line
              aes(x = vo_nest,
                  ymin = lwr, # lower bound
                  ymax = upr), # upper bound 
              alpha = 0.1, # changes transparency of ribbon
              inherit.aes = FALSE) +
  # labels the x and y axis in the table
  labs(x = "Visual obstruction at nest",
       y = "Probability of nest site use") +
  
  theme_classic() # changes theme
